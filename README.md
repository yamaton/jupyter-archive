# jupyterlab-qzv

A Jupyterlab extension to view Qiime2 QZV files. This works **only in JupyterHub running Shiny backend**.

The codebase is a fork of [jupyter-archive](https://github.com/jupyterlab-contrib/jupyter-archive).
